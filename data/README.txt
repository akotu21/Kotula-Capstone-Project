Data files included in the Kotula-Capstone-Project Repository:

gad_cleaned.csv - amphibian survey data from Dr. Ashpole's Global Amphibian Decline course over the last ten years.
moon_illumination_1800-2100.csv - The daily moon illumination and phase calculated by Peter Menzies on Kaggle using a mathematical formula.
gad_moon.csv - an updated version of the gad database where I added moon data from the Kaggle dataset to each amphibian observation.
